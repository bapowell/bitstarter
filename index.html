What can you do with an older, but perfectly good, PC?

For years I've thought about how to "recycle" the functional desktop computers that businesses and individuals get rid of each year. An interesting solution is to utilize the PC as the core of an electronics lab station setup. It has a power supply that can be tapped into to provide benchtop power, and it usually has a sound card that can be utilized for a poor-man's oscilloscope and function generator. Those two items alone are enough for a decent start to an electronics workbench, for learning and playing around with electronics. But with free software (Linux, Python, etc.), the variety of hardware ports on most PCs (serial port, parallel port, USB ports, etc.), and the wealth of knowledge available via the internet, there's practically no limit to the amount of learning and experimentation that can be accomplished with an otherwise free PC.

Linux-based PC (could be a Raspberry PI) running:

  • xoscope (sound card oscilloscope)
  • program for using sound card line-out as a function generator (GNU Radio ?, Snack Sound Toolkit, Ecasound)
  • program for reading game port (switch status and pot/resistance)
  • program for reading/writing parallel port (comedi, which can also interface to 8255 on PC bus) (pport, http://sourceforge.net/projects/pport)
  • program for reading/writing IR (infrared) module
  • program for reading/writing MIDI port (e.g. abc2midi)
  • AVR programming software
  • PIC programming software
  • several editors (edit PIC code, Python code, etc.) - vi, IDLE (Python IDE), ...
  • spreadsheet (electronic formula calculations, graphing of captured data, etc.)
  • Python (to write programs which can read/write the game port (pyGame), parallel port (pyParallel), serial port (pySerial), etc.)

Circuit station component blocks

  • solderless breadboard
  • power supply block
  • potentiometers
  • transistors
  • ...(many more)
